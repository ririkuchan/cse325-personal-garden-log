# Personal Garden Log

A comprehensive web-based application designed for plant enthusiasts to track and manage their gardening activities. This project was developed as a final group project for CSE325, demonstrating proficiency in the .NET development ecosystem.

## 🚀 Features
- **User Authentication**: Secure registration and login system managed via ASP.NET Core Identity.
- **CRUD Operations**: Full functionality to Create, Read, Update, and Delete plant records effortlessly.
- **Responsive Design**: Optimized for a seamless user experience across desktops, tablets, and smartphones.
- **Accessibility & Quality**: Built with a focus on web standards, performance, and accessibility.

## 🛠 Technology Stack
- **Framework**: .NET 8.0 (Blazor Web App)
- **Language**: C#
- **Database**: SQLite (managed by Entity Framework Core)
- **Deployment**: Microsoft Azure App Service
- **Authentication**: ASP.NET Core Identity
